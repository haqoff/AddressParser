Как рассчитать вероятность:
	Полный адрес:
	Область - район - город - улица

	Но в большиснтве случаев:  область - город - улица
	
	Смысл алгоритма: нужно рассчитать чего не хватает.
	1. Вычисляем кол-во всех обьектов int probabilty = кол-во.

	2.Вычисляем сколько обьектов неизвестно(сколько нужно для полного адреса) int needForFullAddress.
	{
		Если нет области, то needForFullAddress++;
		Если нет города, то needForFullAddress++;
		Если нет улицы, то needForFullAddress++;
	}
	
	3.Вычсляем сколько обьектов имеет AOType ==null  int nullObjects = кол-во;

	4.Вычисляем:

		probabilty -= Math.Abs(needForFullAddress - nullObjects);
